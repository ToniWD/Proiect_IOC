# Proiect_IOC

# Configurare mediu Unity

Pentru a rula și edita proiectul, este necesar să folosești versiunea exactă de Unity:

Unity 6000.2.13f1

Asigură-te că ai această versiune instalată din Unity Hub → Installs → Add.

## Extensii recomandate

Dacă folosești Visual Studio ca editor pentru cod, instalează extensia:

Visual Studio Tools for Unity

Aceasta oferă integrare completă între Unity și Visual Studio, inclusiv IntelliSense și debugging.

## Clonarea și deschiderea proiectului

1. Clonează repository-ul local.
2. Deschide Unity Hub și selectează opțiunea Open.
3. Alege folderul „SavePoint-Momo” din proiectul clonat.
4. Unity va descărca automat toate dependințele necesare (Packages etc.).

## Flux de lucru recomandat (Git Workflow)

Pentru orice modificare asupra fișierelor din Unity — cod (.cs), scene (.unity), prefab-uri sau alte resurse — este recomandat să creezi un branch nou înainte de a începe lucrul.

Acest lucru ajută la evitarea conflictelor, menține istoricul clar și permite review adecvat înainte de merge.
